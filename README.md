# Declercq Stortbeton Website

Moderne website voor Declercq Stortbeton - 50 jaar ervaring in stortbeton, granulaten en betonpompdienst.

## Features

- ✅ Interactieve beton calculator
- ✅ Pompbereik visualisatie (24m - 65m)
- ✅ Responsive design (mobile-first)
- ✅ Smooth scroll animaties
- ✅ Hero slider met 4 afbeeldingen
- ✅ Professionele SVG icons
- ✅ Modern design met Tailwind CSS

## Technologie

- HTML5
- CSS3 (Tailwind CSS)
- Vanilla JavaScript
- Geen frameworks of dependencies

## Live Website

Website is live op GitHub Pages: [Bekijk Website](https://jouw-username.github.io/declercq-stortbeton-redesign/)

## Lokaal Draaien

1. Open de `public` folder
2. Start een lokale server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open browser op `http://localhost:8000`
